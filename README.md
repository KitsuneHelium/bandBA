## ✨ 特性

- **完整的视觉小说引擎**：支持文字、立绘、背景、选项、分支剧情
- **沉浸式体验**：打字机效果、自动播放、逐行回退功能
- **完善的存档系统**：自动保存游戏进度和回退历史
- **丰富的动画效果**：背景淡入淡出、角色抖动、屏幕震动
- **自定义设置**：可调节字体大小、打字速度、自动播放间隔
- **跨章节跳转**：支持多剧本无缝切换
- **极致优化**：专为低功耗智能手环设备优化，流畅运行

## 📱 支持设备

- ✅ **小米手环9 Pro**（完美适配，336x480分辨率）
- ⚠️ 其他运行小米Vela系统的设备（理论兼容，未经过测试）

## 🚀 快速开始

### 下载安装

1.  前往 [Releases](sslocal://flow/file_open?url=https%3A%2F%2Fgithub.com%2FHlmSec%2FbandBA%2Freleases&flow_extra=eyJsaW5rX3R5cGUiOiJjb2RlX2ludGVycHJldGVyIn0=) 页面下载最新的安装包
2.  使用 **AIoT Toolkit 2.0** 连接你的手环
3.  点击"安装应用"，选择下载的rpk文件

### 制作剧本

bandBA使用简单的JSON格式编写剧本，将编写好的剧本文件命名为 `gamedata1.txt`，放入 `/Common/GAMEDATA/` 目录即可。
